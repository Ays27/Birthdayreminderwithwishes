Birthday Reminder & Wishes App
A full-stack web application that allows users to manage birthdays and organize personalized wishes. Users can create accounts, add birthday entries, edit or delete them, and view all saved birthdays in a simple dashboard.

Features

User authentication (signup and login)
Add, edit, and delete birthday entries
Store and retrieve data using a PostgreSQL database
Dashboard for viewing all saved birthdays
Highlight birthdays occurring on the current date
Responsive and simple user interface

Tech Stack
Next.js (Frontend and API routes)
TypeScript
PostgreSQL
Drizzle ORM
Tailwind CSS
Vercel (Deployment)

Live Demo
https://birthdayreminderwithwishes.vercel.app

Project Structure
/app – Frontend pages and API routes
/app/api – Backend endpoints (authentication and birthdays)
/db – Database schema and configuration
